# overlay-cursor

Простая утилита для Windows, которая показывает текущую раскладку клавиатуры (`EN`, `RU` и т.д.) рядом с курсором.

## Зачем это нужно

Когда часто переключаешь языки, бывает сложно понять, какая раскладка активна прямо сейчас.
Эта программа выводит небольшой индикатор возле мыши, чтобы это было видно сразу и без лишних действий.

## Что делает программа

- постоянно работает в фоне;
- определяет активную раскладку в текущем окне;
- рисует компактный полупрозрачный оверлей рядом с курсором;
- плавно меняет цвет индикатора при переключении языка.
